### SIGNINT uzdevums

Programma veic *while* ciklu, kas darbojas mūžīgi, lai to pārtrauktu, programma gaida CTRL+C ievadi. Gadījumā, ja kaut kas tomēr nav korekti ir arī Exception bloks.
Piemērs no termināļa: 

```bash
Ctrl+c, lai izietu
Strādā...
Strādā...
Strādā...
Strādā...
Strādā...
Strādā...
Ctrl+c strādā
```
